# Thriend Or Faux

A Chrome extension for [threads.com](https://www.threads.com). Hover a username to see quick stats under the native hover card; click "Analyze posts →" for a Claude-generated read on whether that person seems worth engaging with.

- **Site, install info, and privacy policy:** https://thatadamguy.github.io/thriend-or-faux/
- **Status:** invite-only preview, currently under Chrome Web Store review.
