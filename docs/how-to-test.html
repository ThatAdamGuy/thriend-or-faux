How to test

Quick-start for trusted testers · fuller guide coming soon

1 · Install

Open the Chrome Web Store link you received, using desktop Chrome (or another Chromium browser) signed in with the Google account email you gave us — the listing is private and invisible to anyone else. Click "Add to Chrome."

2 · Add your Claude API key

  • Create a key at platform.claude.com (Settings → API keys) and add a small amount of credit — $5 covers roughly 1,500 profile analyses.
  • Click the Thriend or Faux compass icon in your Chrome toolbar and paste the key in.

Profile stats work without a key; the key is only needed for the AI analysis.

3 · Use it

  • Go to threads.com and hover over any username — a stats card appears under the native hover card.
  • Click "Analyze posts →" to open the side panel with Claude's assessment. The first analysis of a profile takes ~10–20 seconds; repeat lookups are instant (cached for 12 hours — the ↻ refresh link forces a fresh one).

4 · Report back

Anything odd — wrong stats, stuck "fetching…" states, weird verdicts, layout glitches — goes to @ThatAdamGuy. A screenshot plus the profile name involved helps a lot. The known issues page lists what's already on the list.
